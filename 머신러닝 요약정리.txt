머신러닝 - 기계 학습
-> 컴퓨터에게 패턴을 학습시켜서 새로운 예측값을 얻어내는 방법

1) 지도 학습 
- 문제 데이터와 정답 데이터를 컴퓨터에게 주고 패턴을 학습!
- 각각의 정답을 예측해서 나누는 분류
- 실제 수치값을 예측하는 회귀

2) 비지도 학습
- 문제 데이터만 컴퓨터에게 주고 특정 패턴 또는 단순한 레이블 작업 진행
- 다차원 구조를 단순화 시키는 차원 축소
- 같은 또는 비슷한 데이터끼리 묶어주는 군집화

3) 강화 학습
- 인간의 행동 심리학을 기반으로 컴퓨터를 학습시키는 방법.

과대적합 / 과소적합 / 일반화

1. 과대적합이 뭐지?
- 학습을 진행할 때 학습된 패턴이 훈련데이터 셋트에 맞춰져 있다.
-> 새로운 데이터에 반응을 못하는 모습을 보이고 있다.

2. 과소적합은?
- 학습이 제대로 되지않아서 훈련데이터와 평가데이터 모두 반응을 못함

3. 일반화는?
- 훈련데이터도 어느정도 성능이 나오고 새로운 데이터셋트에도 
  반응을 잘한다

KNN(최근접 이웃 알고리즘)
- 유유상종 / 끼리끼리 논다라는 말이 잘 어울리는 모델
- 새로운 데이터 주변의 값을 확인해서 데이터를 예측하는 모델

Decision Tree(의사결정나무)
- 스무고개와 유사
- 각 노드별 질문을 던지고 yes / no로 나눠가는 방식

분류 평가 지표 - 정확도 / 정밀도 / 재현율 / F1 score

교차검증 - 학습과 평가를 1번씩밖에 안한다. - 못믿겠다.
-> 전체 데이터를 가지고 검증을 진행해서 예상되는 일반화 점수를 뽑아보자

그리드 서치
-> 하이퍼 파라미터는 너무 많다.
-> 최적의 조합식을 찾아내는 방법

앙상블 모델
- 보팅
-> 서로 다른 여러개의 모델을 이용하는 방법
- 베깅
-> 서로 같은 여러개의 모델을 이용하는 방법 - 랜덤 포레스트
- 부스팅
-> 서로 같은 여러개의 모델을 순차적으로 이용하는 방법

정리 : 베깅 / 부스팅은 무슨기반 모델일까? - 트리모델을 기반으로 사용

회귀 모델 - 중요 / 엄청 중요 / 그냥 중요
1. 그럼 회귀가 뭔데요? -> 실제 수치값을 예측!

2. 그럼 선형 회귀는요? -> 실제 수치값을 예측! 뭘 이용해서? 하나의 직선을 사용해서

회귀의 평가 지표 : MSE / RMSE / R2 score / MAE / MAPE
MSE : 평균제곱오차
RMSE : 평균제곱근오차
R2 : 결정계수
MAE : 평균절대오차
MAPE : 평균절대비율오차

경사하강법
-> MSE(평균제곱오차)가 가장 작은 곳을 찾아갈거다!
-> 어떻게? 가중치 w와 편향 b를 수정해서

y=wx+b

규제
- L1-norm : 라쏘 - 직접 상수를 빼주는 방법. 특성 선택이 자동으로 일어남
- L2-norm : 릿지 - 비율로 가중치를 규제하는 방법. 특성 제외가 없다.

로지스틱 회귀
- 이름은 회귀이지만 0과 1로 분류해주는 이진 분류 모델
